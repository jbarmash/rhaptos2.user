Testing¶

runtests¶

This is a script (not setup as as an entry point, probably needs to be) to execute tests in :rf:`rhaptos2.user`

useage:

Run all tests with the given config

python runtests.py –config=../../../local.ini

Run only the test modules in modules

python runtests.py –config=../../../local.ini

Prepopulate (does not build tables)

python runtests.py –config=../../../local.ini –prepopulate=Y

this version will connect to the DB in local.ini, and (assuming the tables exist) will insert through psycopg2 five test users.

(OK I swapped out the sqlalchemy prepopulation (which creates tables too) so the whole table creation thing is a little awkward and needs fixing)
